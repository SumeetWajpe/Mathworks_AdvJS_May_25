Each single-property transition describes the transition that should be applied to a single property or all properties. It includes: zero or one value representing the property or properties to which the transition should apply. This can be set as: A representing a single property. The special value all, which specifies that the transition will be applied to all properties that change as the element changes state. No value, in which case a value of all will be inferred and the specified transition will still apply to all changing properties. zero or one value representing the easing function to use
